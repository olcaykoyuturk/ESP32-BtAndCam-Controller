<h1>ESP32 CAM CONTROLLER</h1>
<p>My app works by sending char over Bluetooth. It is also connected to esp32cam via ip.</p>
<br>
<h2>Commands</h2>
<li>Forward: 'F'</li>
<li>Left: 'L'</li>
<li>Backward: 'B'</li>
<li>Right: 'R'</li>
<li>Speed 1: 'A'</li>
<li>Speed 2: 'B'</li>
<li>Speed 3: 'C'</li>
<li>Slider: '0','1','2','3','4'</li>
<br>
